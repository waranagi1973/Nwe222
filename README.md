## 注意書き
* このプロジェクトは、micro:bit V2のBasicセットで作成されたものです。
  つまり、micro:bit本体がV2のものでないと動作しませんので十分にご注意ください。
* 起動するのに約8秒かかります。

## 使用方法
* Aボタンで方角を表示します。
* Bボタンで温度(気温)を表示します。
* A&Bボタンで赤い懐中電灯を実行します。
*ロゴに触ると実行中のモードを終了します
## 方角
* 0度=北
* 90度＝東
* 180度=南
* 270度=西
